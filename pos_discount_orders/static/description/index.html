POS: Lineas de orden con descuento

Una simple manera de aplicar descuento a los articulos de una orden desde el punto de venta.

El módulo permite realizar descuentos para todas las líneas de pedido con un valor de descuento único.

Simplemente seleccione el botón de descuento y escriba descuento en el teclado numérico como de costumbre. El módulo aplicará descuento para todos los artículos del pedido.
